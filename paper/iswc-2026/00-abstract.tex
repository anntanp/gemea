GEMEA is a publicly accessible knowledge graph over 27M objects from the German Digital Library (DDB), served via a QLever SPARQL endpoint, browsable through SHMARQL, and queryable by AI agents via an Model Context Protocol (MCP) interfaces. It is the first open structured access layer for the DDB corpus. The resource includes a MOCHO-based ontology alignment layer mapping DDB metadata, described using the Europeana Data Model (EDM), to several domain- and modality-specific ontologies, developed iteratively against corpus analysis rather than projected onto a pre-existing fixed schema, with partial linking to the Gemeinsame Normdatei (GND), the German Integrated Authority File. By exposing the DDB metadata corpus as a SPARQL-queryable knowledge graph with ontology alignments and partial GND linking, GEMEA enables aggregate, structural, and federated queries that were not possible previously. The Model Context Protocol (MCP) interface further opens the corpus to agentic workflows, allowing LLM-based research assistants to retrieve and reason over DDB metadata. The resource is intended to support research on ontology evaluation, agentic knowledge graph applications, and provenance modelling of LLM-assisted enrichment.
