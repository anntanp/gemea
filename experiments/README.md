# experiments/

Evaluation runs and notebooks used for the paper's quality and validation sections.

## Conventions

- One subdirectory per experiment: `YYYY-MM-DD_description/`
- Each experiment directory contains:
  - `README.md` — hypothesis, method, results, conclusion
  - Input data references (paths or download instructions)
  - Scripts or notebooks to reproduce the run
  - Output files (metrics, logs, figures)
- Results must be reproducible from the README instructions alone.
